POpSiCLE

Detailed Description

Gauss legendre related routines.

Author
Alejandro de la Calle. Queen's University Belfast.
Daniel Dundas. Queen's University Belfast.
Date
24/09/2015

This module contains routines to prepare and perform Gauss-Legendre quadratures. Also, it calculates Legendre polynomials, Clebsch-Gordon coeffcients and spherical harmonics transforms.

This subroutine calculates the log of factorials. The log of the factorials is chosen since we can then use addition/subtraction instead of multiplication/division when calculating quantities involving factorials. This gives more stability.

This subroutine calculates the Legendre polynomials for all permissible {il,im} combinations which are given by il={0,MAXL} => im={0,il}. This routine is based on that given in Numerical Recipies in FORTRAN, 2nd Edition, page 246-248, in which the Legendre polynomial was calculated as a function value using a recurrence relation. This routine improves this relation by storing the intermediate Legendre polynomials which the recurrrence relation discarded.

Parameters
[out]legenplThe array storing the Legendre polynomials given by: legenpl(x, im, il)
[in]costhetaThe array storing the cosine of the angle at which the Legendre polynomial is calculated at, i.e. x={-1,1}
[in]MAXLThe maximum value of the single electron angular momentum quantum number.
[in]MAXTHETAPTSThe maximum theta point.
